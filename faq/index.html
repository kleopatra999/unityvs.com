F.A.Q. frequently asked questions

1. What version of Visual Studio does Visual Studio Tools for Unity support?

Visual Studio Tools for Unity supports Visual Studio 2010, Visual Studio 2012 and Visual Studio 2013. You need to have a fully functional Visual Studio. The minimum SKU Visual Studio Tools for Unity supports is Visual Studio Professional.

2. Does Visual Studio Tools for Unity support the Visual Studio Community Edition?

Yes it does!

3. What version of Unity does Visual Studio Tools for Unity support?

Visual Studio Tools for Unity requires Unity 4 or above on Windows. Visual Studio Tools for Unity works with both the Free and the Pro version of Unity.

4. What programming language does Visual Studio Tools for Unity support?

Visual Studio Tools for Unity supports C# 3.

5. Is Visual Studio Tools for Unity localized in any language?

Visual Studio Tools for Unity is currently only available in English.
