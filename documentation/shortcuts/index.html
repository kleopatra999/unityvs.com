Shortcuts

You can configure VSTU commands shortcuts, using Tools/Options/Environment/Keyboard. Here are the default commands and shortcuts we use:

MonoBehavior Wizard (in a code window)
EditorContextMenus.CodeWindow.ImplementMonoBehaviours: CTRL+SHIFT+M

Quick MonoBehavior Wizard (in a code window)
EditorContextMenus.CodeWindow.QuickMonoBehaviours: CTRL+SHIFT+Q

Attach to Unity Debugger (player or editor)
Debug.AttachUnityDebugger: NONE

Access the Unity documentation
Help.UnityAPIReference: CTRL+ALT+M, CTRL+H

Open the Unity Project Explorer
View.UnityProjectExplorer: SHIFT+ALT+E
