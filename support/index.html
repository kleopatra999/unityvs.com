Support

We are doing our best to make sure that Visual Studio for Unity works well. Nevertheless, should you face any issue, please don't hesitate to get in touch with us and get support.

Make sure to visit the list of known issues on the documentation. If you want to report a bug, please use the Microsoft Connect site. If you have a suggestion, please use the UserVoice site.

Visual Studio customesr seeking support may also visit the Visual Studio Support site.
